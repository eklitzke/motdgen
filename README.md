# motdgen

Generate dynamic MOTD: Message of the day

## Setup

```
sudo python setup.py install
sudo echo "session    include      motdgen" >> /etc/pam.d/sshd
sudo motdgen-cache-dnfupdateinfo
```

or, just

``[sudo] dnf install fedora-motd``
